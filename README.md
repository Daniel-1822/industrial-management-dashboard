**Soonfield** is a comprehensive, high-performance industrial management ecosystem designed for the modern factory. Built with a "Cyber-Industrial" aesthetic, it provides a unified interface for administration, supply chain tracking, and logistics optimization.

## üé® Aesthetic & Design System

Soonfield features a unique **Cyber-Industrial** design language:
- **Primary Industrial**: `#FFC500` (Safety Yellow) - Used for Administrative modules.
- **Logistics Success**: `#10B981` (Emerald Green) - Used for Transport modules.

- **UI Elements**: Scanlines effect, diagonal safety stripes, and high-contrast typography.

## üõ†Ô∏è Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Daniel-1822/industrial-management-dashboard.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Custom Developer Scripts
- `npm run gen`: Create a new component template automatically.
- `npm run page`: Create a new page template.
